How does the Internet work?

HTML

Hyper Text Markup Language

HTML Tag

A term in between two opposing angle brackets


    In Pairs                     Or Single
    <tag> </tag>                 <tag>

HTML element

is tag + content


    <tag>
      This is some content inside of a tag
    </tag>

Tag Attribute

Attributes assign a property to a element


    <tag attr="value">
      Content
    </tag>

Contrived Example


    <dog size="big" color="red"> Clifford </dog>

    **Please note this is not real HTML

CSS

Cascading Style Sheets

Selector

specify an element to apply styles to

p { color : tomato }

declaration (or "style")

the thing we want to do to the selected elements, made up of property and value

p { color : tomato }

Property

the thing I want to change about my elements

p {color : tomato }

Value

What I want to change about my elements

p { color : tomato}

Selectors

targeting your markup

Type Selector

Use the tag name


      h1 {
        font-size : 100px;
        font-weight : bold;
      }

      p {
        color : #f0f;
        padding : 15px
      }

      ul {
        color : green;
      }

Class Selector

Add a class attribute to a tag and select on that.


    <!-- HTML -->
    <div class="my-div"> blah blah blah </div>

    /* CSS */
    .my-div {
      font-size : 100px;
      font-weight : bold;
    }
